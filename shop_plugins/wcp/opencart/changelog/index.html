Change log

Change log

for OpenCart and Wirecard Checkout Page

Version 2.0.1 (2018-03-18)

  • Calculate device fingerprint only when Ratepay is enabled for invoice and installment.

Version 2.0.0 (2017-11-15)

  • Add Opencart 3.0 shop compatibility.

  • Add PHP 7.1 compatibility

  • Remove payment method @Quick.

Version 1.5.2 (2017-09-25)

  • Rename payment method SOFORT Banking to Sofort./Online bank transfer.

Version 1.5.1 (2017-08-28)

  • Add new parameter consumerDeviceId.

Version 1.5.0 (2017-06-14)

  • Implement shopping basket.

  • Update parameters customerStatement and orderReference.

  • Implement payment methods Invoice by Wirecard and Invoice:RatePAY.

  • Implement payment method Installment: RatePAY.

  • Implement payment method Masterpass.

  • Update pre-checks for payment methods Installment and Invoice.

  • Update payment methods, remove deprecated payment methods.

  • Add financial institutions for payment methods eps-Überweisung and iDEAL.

  • Update translations, set default template for payment init.

  • Print server error messages.

Version 1.4.1 (2017-04-18)

  • Improve order update for pending payments.

  • Update permission path for validation.

Tested version: OpenCart 2.3.0.2 with PHP 5.6

Version 1.4.0 (2016-11-29)

  • Add new parameter for payment method paysafecard.

  • Improve security on payment server response.

Tested version: OpenCart 2.3.0.2 with PHP 5.5 and PHP 5.6

Version 1.3.0 (2016-11-14)

  • Add OpenCart v2.3.0.2 compatibility.

  • Update file structure and paths.

  • Add/Update terms of use.

  • Update failure redirection.

Tested version: OpenCart 2.3.0.2 with PHP 5.5

Version 1.2.0 (2015-04-27)

  • Plugin updated and improved.

  • New payment methods ePay.bg, TatraPay, and Voucher added.

  • Mobile layout detection implemented.

  • Compatibility to OpenCart 2.0.1.1 added.

  • Payment method renaming from "Mobile Phone Invoicing" to "paybox".

  • Admin controller and view refactored.

Version 1.1.0 (2013-08-01)

  • Product renaming from QPAY to Wircard Checkout Page.

  • Added new payment methods Bancontact/Mister Cash, CLICK2PAY, eKonto, Installment, Instant Bank, Invoice, Moneta.ru, mpass, Przelewy24, POLi, Skrill Direct and Skrill Digital Wallet added.
