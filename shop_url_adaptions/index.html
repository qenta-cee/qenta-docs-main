Plugin Shop URL Adaptions

Plugin Shop URL Adaptions

Merchants which are using our shop plugins have to switch the endpoint from Wirecard to new hobex Service IT API.

Old endpoint New endpoint

checkout.wirecard.com

papi.hobex.at

secure.wirecard-cee.com
Change only the first part of the URL, everything after .com/ stays the same!
Table 1. Example of switching endpoint
From To

https://checkout.wirecard.com/page/init.php

https://papi.hobex.at/page/init.php

https://checkout.wirecard.com/seamless/frontend/init.php

https://papi.hobex.at/seamless/frontend/init

Choose the shop system in the list below and change the endpoint accordingly.
