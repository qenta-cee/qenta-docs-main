Secret and fingerprint

Secret and fingerprint

What is a "secret"?

A secret is a pre-shared-key (see Wikipedia) which is only known by the merchant, the integrator of the web shop and QENTA Payment CEE. This secret is used to secure the transfer of all sensible parameters and their values between your web shop and the QPAY Checkout Page.

 To ensure the secure communication it is essential that you NEVER show or give this pre-shard-key to others not involved in developing the web shop!
Additionally please NEVER send this pre-shared-key by mail, e-mail or instant messaging and ensure also when it is send by fax that only authorized persons have access and can read this fax!

Please do also NEVER send the secret as a parameter to the QPAY Checkout Page!
If you have suspicion that your pre-shared-key may be known by unauthorized persons, please contact our Support team immediately via support@qenta.com to request the creation and submission of a new secret to you.

What is a "fingerprint"?

A fingerprint is a method to ensure that sensible parameters and their values sent from your web shop to the QPAY Checkout Page and vice versa are not changed by some- or anyone which is transferring the data over the Internet.

A fingerprint is created by adding all parameter values which have to been secured to a string, then also adding the value of the secret (aka pre-shared-key) to this string. Finally this string is hashed by an algorithm (we use HMAC SHA 512) and this results to a fingerprint.

When you submit data from your web shop to the QPAY Checkout Page you also send the fingerprint and the name and order of all parameters used for creating the fingerprint (fingerprint order). The QPAY Checkout Page then creates also the fingerprint of all received parameter values including your merchant specific secret stored in the QPAY Checkout Platform. If the fingerprint you send us and the fingerprint computed on the QPAY Checkout Page are the same, the values of your sent parameters are not modified by someone like a man-in-the-middle attack (for details have a look at Wikipedia).
