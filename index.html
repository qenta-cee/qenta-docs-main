The hobex Service IT Payment Solutions

The payment page for a webshop

Easy and quick to integrate, secure and convenient

Checkout Page

  • Easy to integrate in a webshop
  • Customer gets redirected to Checkout Page for payment process
  • Customization via CSS
  • Available in 26 languages
  • Connection to shop systems and payment plugins available
  • Backend Operations available
  • PCI DSS SAQ tier: A

Direct and flexible integration for a webshop

Integration of a seamless form in the webshop

Checkout Seamless

  • Integrate the checkout for Credit Card and SEPA directly into the webshop without redirect for the customer
  • Entry of card data and authorization can take place at separate times
  • Alternative payments will be redirected to the respective acquirer
  • Customization via CSS
  • Available in 26 languages
  • Backend Operations available
  • PCI DSS SAQ tier: A or A-EP (depends on the integration)

Solution for written and telephone orders

Process all payment data provided via mail or telephone immediately

Checkout Terminal

  • Easy to use web-based solution for a call center
  • Login to the website hosted by hobex with user name and password
  • Credit card, SEPA and recur payments are available
  • Verification of payment data in real-time
  • PCI DSS SAQ tier: C-VT

Solution for all written and telephone orders that are not real-time critical

Ideally suited for all applications that are not real-time critical (catalog orders, recurring payments, subscriptions, from account interfaces…)

Checkout Automated

  • Web-based solution for uploading multiple payment data via batch-file
  • Login to the website hosted by hobex with user name and password
  • Credit card, SEPA and recur payments are available
  • Verification of payment data in real-time after upload
  • PCI DSS SAQ tier: C-VT

Administrative dashboard application for payment link creation and management

Advantage of a simple graphical interface to create payment links

PayByLink

  • Web-based solution to create/manage a checkout link and QR-code
  • No webshop needed
  • Dashboard for transaction management
  • Solution to offer a secure payment option for the B2C or B2B customer
  • Let the customer choose the preferred payment method
  • PCI DSS SAQ tier: A

API solution for payment link creation and management

Create an own PBL-solution based on merchant needs

API PayByLink

  • API solution to create/manage a checkout link and QR-code
  • No webshop needed
  • Optional dashboard for transaction management
  • Best solution for the integration in the ERP
  • PCI DSS SAQ tier: A

Server-to-server solution for enabling process transactions from any source

Checkout Enterprise enables a seamless link between a online shop, inventory management, complaint and accounting tools and any third-party systems

Checkout Enterprise

  • Flexible server-to-server integration
  • Frontend provided by merchant according to the needs
  • 100% control of each step in the payment process
  • Best solution not only for merchants but also for partners where a customized solution is preferred
  • PCI DSS SAQ tier: D
