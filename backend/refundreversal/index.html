refundReversal

This operation cancels a credit note and can only be done before the corresponsing day-end closing has been executed and the financial service provider supports this operation.

The required order of the parameter values when computing the fingerprint is: customerId, shopId, toolkitPassword, secret, command, language, orderNumber and creditNumber

Additional required request parameters

Parameter Data type Short description

orderNumber

numerical with a fixed length of 9

order number of payment

creditNumber

numeric with a fixed length of 9

credit number which has to be cancelled
