JAPI Version 4.8 Update

JAPI Version 4.8 Update

To keep up with the latest technology and developments and to ensure the successful implementation of new features and payment methods in your online shop, we strongly recommend updating to the current JAPI version 4.8, as this version will be further developed.

The following step-by-step instructions provide the necessary information to guide you through the migration process.

Download

  • JAPI 4.8

  • TrustStore file

Update guide for JAPI version 4.x to 4.8

Requirements

  • Java version 1.7 or higher.
    We recommend to install Java 1.8.

Changes

Endpoint
QTillSettings settings = QTillSettings(new URL("https://checkout.wirecard.com/enterprise"));
Authentication
settings.setBasicAuthUsername("<keystore filename without path and extension>");
settings.setBasicAuthPassword("<keystore password>");
Truststore
settings.setTruststoreFile(new File("wce-trust.jks"));
settings.setTruststorePassword("<truststore password>");

Updating usage of removed giropay constructor
Usage of removed constructor
QTillInstance qtill; // instantiate your qtill instance here
int orderNumber = qtill.generateOrderNumber();
LOG.info("orderNumber: {}", orderNumber);

URL returnUrl = new URL(RETURN_URL);
QTillNumericToken bankNumber = new QTillNumericToken(BANK_NUMBER);
QTillNumericToken bankAccount = new QTillNumericToken(BANK_ACCOUNT);
QTillGiropayInitiation giropay = new QTillGiropayInitiation(
    ACCOUNT_OWNER, bankNumber, bankAccount, returnUrl, WINDOW_NAME
);

QTillOrderReference orderReference = new QTillOrderReference(orderNumber, "Order Description");
QTillAmount amount = new QTillAmount("1.23", new QTillCurrency("EUR"));
qtill.initiatePayment(orderReference, amount, giropay);

QTillPayment payment = qtill.verifyPayment(orderNumber);
LOG.info("paymentState: {}", payment.getState());
Usage of replacement constructor
QTillInstance qtill; // instantiate your qtill instance here
int orderNumber = qtill.generateOrderNumber();
LOG.info("orderNumber: {}", orderNumber);

QTillNumericToken bankNumber = new QTillNumericToken(BANK_NUMBER);
URL returnUrl = new URL(RETURN_URL);
QTillNumericToken bankAccount = new QTillNumericToken(BANK_ACCOUNT);
QTillGiropayInitiation giropay = new QTillGiropayInitiation(bankNumber, returnUrl, WINDOW_NAME);
giropay.setAccountOwner(ACCOUNT_OWNER);
giropay.setBankAccount(bankAccount);

QTillOrderReference orderReference = new QTillOrderReference(orderNumber, "Order Description");
QTillAmount amount = new QTillAmount("1.23", new QTillCurrency("EUR"));
qtill.initiatePayment(orderReference, amount, giropay);

QTillPayment payment = qtill.verifyPayment(orderNumber);
LOG.info("paymentState: {}", payment.getState());
