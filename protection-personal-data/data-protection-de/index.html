Datenschutz Info

Überblick

Auf dieser Seite erhalten alle relevanten Informationen rund um das Thema Verarbeitung personenbezogener Daten durch Produkte der hobex Service IT.

Following information in English: Switch to English.

hobex Service IT Produkte und Datenverarbeitung

Im Rahmen der Zahlungsabwicklung und abhängig vom jeweiligen Produkt werden von verschiedenen Stellen (z.B. Merchants, Endkunden) sowohl bestimmte Daten an hobex Service IT übermittelt (Input-Daten), wie auch hobex Service IT wiederum bestimmte Daten an diese Stellen zurück übermittelt (Output-Daten).

Input Daten

hobex Service IT erhält Daten von drei verschiedenen Stellen:

  • Daten, die vom Zahlungsauslösenden je Produkt eingegeben werden (z.B. Dateneingaben des Endkunden im Zahlungsfenster)

  • Daten, die vom Merchant an die hobex Service IT Zahlungsschnittstelle gesendet werden (Request-Parameter)

  • Plugin-spezifische Daten des Online-Shops

Output Daten

hobex Service IT verarbeitet die Zahlungstransaktion und übermittelt folgende Daten zurück:

  • Daten an den Merchant u.a. über das Ergebnis der Zahlungstransaktion (Response-Parameter)

  • Feature-Spezifische Daten

Datenschutzhinweise zu einzelnen Produkten

Nachfolgend ist über Detailansichten zu einzelnen Produkten ersichtlich, welche Daten von welchem Produkt (inklusive Zahlungsmittel und/oder Zusatzfeatures) als Input- und Output-Daten verarbeitet werden. Sämtliche der darin angezeigten Parameter müssen verarbeitet werden, um den Vertragsgegenstand gegenüber dem Merchant erbringen zu können (Zweck der Datenverarbeitung):

  • Checkout Page: Datenschutzhinweise Checkout Page

  • Checkout Seamless: Datenschutzhinweise Checkout Seamless

  • Checkout Automated: Datenschutzhinweise Checkout Automated

  • Checkout Terminal: Datenschutzhinweise Checkout Terminal

Datenschutzhinweise zu einzelnen Parametern

Für die Erfüllung des Vertragsgegenstands bzw für die Abwicklung einer Zahlungstransaktion müssen eine Vielzahl an Daten (Parametern) sowohl vom Merchant oder dem Endkunden an die hobex Service IT Zahlungsschnittstelle übermittelt werden (sog. Input-Daten), also auch wieder von hobex Service IT an diese zurück gesendert werden (sog. Output-Daten).

Für all jene Parameter, die ein personenbezogenes Datum darstellen und im Rahmen dieser Zahlungsabwicklung verarbeitet werden, wird auf nachfolgender Seite angegeben, worin der Personenbezug besteht, wer die betroffene Person ist, um welche Datenkategorie es sich handelt und worin der Zweck der Datenverarbeitung begründet liegt:
