Datenschutzinfo für Checkout Automated

Datenschutzinfo für Checkout Automated

Auf dieser Infoseite können Sie sich über alle Daten informieren, die das Produkt Checkout Automated verarbeitet.

Die Datenverarbeitung von Checkout Automated ist umso umfangreicher, je größer der vom Merchant bestellte Produktumfang ist. Bei der Bestellung des Produkts Checkout Automated kann unterschieden werden zwischen

  • 1 Basisprodukt, optional erweitert um

  • 1 bis N Zahlungsmitteln, und/oder

  • 0 bis N optionalen Zusatzfeatures

Die allgemeine Funktionsweise des Produkts Checkout Automated-sowohl in seiner Basisversion als auch abhängig vom Einsatz verschiedener Zahlungsmittel-ist zudem hier ausführlich beschrieben: Bedienungsanleitung Checkout Automated

Eingaben Merchant

Sämtliche hier angegebenen personenbezogenen Parameter müssen verarbeitet werden, um das Produkt Checkout Automated in seiner Basisversion dem Merchant zur Verfügung stellen zu können (Zweck der Datenverarbeitung).

Die hier angezeigten Parameter unterteilen sich in Parameter

  • die unabhängig vom Zahlungsmittel verarbeitet werden (Standard)

  • die abhängig vom jeweiligen Zahlungsmittel verarbeitet werden (Zahlungsmittel)

Status der Zahlung als CSV-Datei herunterladen (Output)

Nach dem Hochladen der Zahlungsdaten Ihrer Endkunden, wird die Datei verarbeitet. Sobald die Verarbeitung abgeschlossen ist, können die Transaktionen als CSV-Datei heruntergeladen werden, welche Sie wieder in eine Excel-Datei umwandeln können. Dadurch wird jedoch keine zusätzliche Verarbeitung von personenbezogenen Daten vorgenommen.

Zusatzfeatures

Für jedes zusätzlich zum Basisprodukt Checkout Automated bestellte Feature ist hier ersichtlich, welche Daten das jeweilige Feature verarbeitet.

Feature Verarbeitete Daten und Datenschutzinfos

1. Weitere Zugänge zum hobex Service IT Payment Center

Für die Web-Applikationen des Händlerlogins wird ein neuer Zugang für den Merchant angelegt, der nicht personengebunden ist. Für den Zugang ist ein Benutzername und ein Passwort erforderlich. Dadurch kommt es zu keiner zusätzlichen Verarbeitung von personenbezogenen Daten.

2. Zusätzliche Währungen

Abhängig vom Produktkatalog des Merchants sind bestimmte Produkte in 1 bis N Währungen zahlbar, die von hobex Service IT als Konfigurationseinstellungen hinterlegt werden. Dadurch kommt es zu keiner zusätzlichen Verarbeitung von personenbezogenen Daten. Allerdings ist die Währung der Zahlung selbst ein personenbezogenes Datum, die über die Währung der abgewickelten Zahlungstransaktion des Merchants Auskunft gibt.

3. Wiederkehrende Zahlung

Mit diesem Feature wird der Merchant über eine Konfigurationseinstellung seitens hobex Service IT in die Lage versetzt, wiederkehrende Zahlungen durchführen. Für die Durchführung einer wiederkehrenden Zahlung ist das Batch-File wie in der Checkout Automated-Dokumentation beschrieben auszufüllen. Dadurch kommt es zu keiner zusätzlichen Verarbeitung von personenbezogenen Daten.
