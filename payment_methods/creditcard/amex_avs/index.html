American Express - Address Verification System (Amex AVS)

American Express - Address Verification System (Amex AVS)

If you want to use this feature offered by American Express, please contact our support teams for enabling the additional response parameters for your account.

Parameter State Data type Short description

avsResponseCode

Success

Alphanumeric including special characters with a variable length of up to 5 characters.

Response code of AVS check.

avsResponseMessage

Success

Alphanumeric including special characters with a variable length of up to 256 characters.

Response text of AVS check.

avsProviderResultCode

Success

Alphanumeric including special characters with a variable length of up to 5 characters.

Provider result code of AVS check.

avsProviderResultMessage

Success

Alphanumeric including special characters with a variable length of up to 256 characters.

Provider result text of AVS check.

Please note that the above mentioned response parameters are enabled by default when you use Wirecard Checkout Seamless in your online shop.

Additionally please ensure to use the following request parameters needed for Amex AVS:

  • Consumer billing data

    • consumerBillingFirstname

    • consumerBillingLastname

    • consumerBillingAddress1

    • consumerBillingAddress2

    • consumerBillingCity

    • consumerBillingCountry

    • consumerBillingZipCode

    • consumerEmail

Detailed description of parameters

avsResponseCode

The parameter avsResponseCode returns the response code of the AVS check.

avsResponseMessage

The parameter avsResponseMessage returns the response text of the AVS check.

avsProviderResultCode

The parameter avsProviderResultCode returns the result code of the AVS check as sent by the provider, i.e. American Express.

avsProviderResultMessage

The parameter avsProviderResultMessage returns the result text of the AVS check as sent by the provider, i.e. American Express.
